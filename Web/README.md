## 目录结构说明

本目录包含 Web 版 VolcEngineRTC 的所有快速开始源代码

```
├─ Web
│  │─ QuickStart_Demo           // 快速开始
│  │  │─ ReactDemo              // 基于React
│  │  │  ├─ public              // 图片等静态资源
│  │  │  ├─ src                 // 进房、通信等基础模块
│  │  │  ├─ README.md           // 项目跑通指南
│  │  │─ BasicDemo              // 基于原生JS
│  │  │  ├─ assets              // 图片等静态资源
│  │  │  ├─ lib                 // vrtc sdk
│  │  │  ├─ README.md           // 项目跑通指南
│  │─ Advanced_Demo             // 进阶能力
│  │  ├─ Example                // Web 页面
|  |  |  ├─ aians               // AI 降噪能力
|  |  |  ├─ beauty              // 美颜能力
|  |  |  ├─ liveTranscode       // 推流到直播能力
|  |  |  ├─ virtualBackground   // 虚拟背景能力
│  │  ├─ h5                     // H5 移动端页面
|  |  |  ├─ beauty              // 美颜能力
|  |  |  ├─ liveTranscode       // 推流到直播能力
|  |  |  ├─ virtualBackground   // 虚拟背景能力
│  │  ├─ README.md              // 项目跑通指南
```
